# INTOPS FMS 품질팀 업무현황

회사 모델/프로젝트의 일정, 체크리스트, 메모를 통합 관리하는 웹 애플리케이션입니다.

## 주요 기능

- 📊 **대시보드** — 모든 카테고리의 진행 현황을 한눈에
- 📡 **상시 모니터링** — 치수 측정/신뢰성 시험/환경 유해물질 등 지속 추적 항목
- 📦 **모델별 진행 관리** — 각 모델/프로젝트별 일정 & 체크리스트
- 📅 **일정표** — 특정일 / 기간 선택, 상태(대기/진행/완료/지연) 관리
- ✅ **체크시트** — NO/대제목/세부진행/목표일/상태/비고 테이블형
- 📝 **메모장** — 자동 저장, 마지막 수정 시각 기록
- 🖨 **인쇄 지원** — 깔끔한 인쇄 레이아웃
- 🕐 **네이비즘 시계 연동**
- 🔄 **실시간 자동 갱신** (5초 폴링)

## 기술 스택

- **백엔드**: Node.js + Express + JSON 파일 저장 (네이티브 의존성 없음)
- **프론트엔드**: Vanilla HTML/CSS/JS (빌드 단계 없음)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 서버 시작
npm start
# → http://localhost:3000
```
